# 💻 Sistema de Cadastro em C

Projeto simples em linguagem C que permite cadastrar, listar e buscar nomes utilizando vetor e busca sequencial.

## 🛠️ Funcionalidades

- Cadastro de nomes
- Listagem de dados
- Busca sequencial

## 🎯 Objetivo

Praticar lógica de programação e estruturas básicas de dados.

## ▶️ Como executar

```bash
gcc main.c -o programa
./programa
